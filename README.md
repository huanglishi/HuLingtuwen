## 简介

采用java的springboot、springcloud、nacos分木块开发，方便多人、及后面持续开发升级开发功能模块，互不影响前面功能使用。

## 特性

- **sprigcloud**： 微服务集成
- **springboot**: java开发框架
- **MyBatis-Plus**：，在 MyBatis 的基础上只做增强不做改变，为简化开发、提高效率而生。
- **国际化**：内置完善的国际化方案
